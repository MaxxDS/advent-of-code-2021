## How to

### 1. Install dependencies

```shell
yarn
```

### 2. Generate a day files

```shell
yarn generate
# or
yarn generate --day 01
```

### 3. Run a single day

```shell
yarn start
# or, to keep process open and listen to changes
yarn dev --day 01
```

### 4. Mark day as complete

```shell
yarn complete
# or
yarn complete --day 01 --stars 2
```
